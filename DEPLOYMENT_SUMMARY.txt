DEPLOYMENT STEPS
================================================================================

Option 1: Vercel Auto-Deployment (Recommended)
  1. Repository is already on GitHub: Rahul93102/frontend-Neosharx
  2. Go to https://vercel.com
  3. Click "New Project"
  4. Select GitHub repository
  5. Configure:
     - Root Directory: frontend
     - Build Command: npm run build
     - Output Directory: dist
  6. Click Deploy
  7. Website live at: https://frontend-neosharx.vercel.app

Option 2: Manual Deployment
  1. Build locally: cd frontend && npm run build
  2. Upload dist/ folder to web server
  3. Configure web server for SPA (fallback to index.html)

Option 3: Using Vercel CLI
  1. npm install -g vercel
  2. cd frontend
  3. vercel
  4. Follow prompts
